Flat VBControls Extension ReadMe!

 

Regardments to this Example
  To execute this example you will need the SSubTmr6.dll file from VBAccelerator (www.vbaccelerator.com), that contains the support for the Controls SubClassing  used by this program. This file is included in the ZIP file.

To register SSubTmr6.dll, open a MS-DOS Box, go to your Windows System directory and write

RegSvr32.exe <path>\SSubTmr6.dll

where <path> is the directory where SSubTmr6.dll is.

For this project, the MSComCtl.cox file has been used to demonstrate the subclassing of the ImageCombo, but there is no need for it on your own projects.

What your going to learn with this Example
 
  • How to make your controls flat, like the ones in the Office Toolbar.
  • How to SubClass controls and Windows.
  • How to launch any file with it's associated program by using the windows API.
  • A full serie of API functions and how to use them.
